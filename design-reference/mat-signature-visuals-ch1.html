✖️➗ Multiplication and Division — Sign Rules

+ × + = +
− × − = +
+ × − = −
− × + = −

Same rules apply for division

⚖️ Four Arithmetic Laws

Commutative Law
a + b = b + a
a × b = b × a
Associative Law
(a + b) + c = a + (b + c)
(a × b) × c = a × (b × c)
Distributive Law
a × (b + c) = a × b + a × c
Identity Law
a + 0 = a   a × 1 = a
a + (−a) = 0

🧮 Problem-Solving — 4 Steps (Fractions Example)

"A quiz has 20 questions. +2 for correct, −½ for incorrect. Total score for incorrect answers was −4. Find the total score."

1
Understanding the Problem
  • Correct = +2, Incorrect = −½
  • Score for incorrect answers = −4
2
Devising a Plan
  • Find number of incorrect answers via division
  • Find total score via multiplication and addition
3
Implementing the Strategy
  • Incorrect answers = −4 ÷ (−½) = 8
  • Total score = (20−8) × 2 + (−4) = 20
4
Doing Reflection
  • 12 correct (24) − 4 (incorrect) = 20 ✓
